## Actions

### Write Log Message

| Input       | Comments                                                                         | Default |
| ----------- | -------------------------------------------------------------------------------- | ------- |
| Log Level   | The log level to use for this log line. Options are: debug, info, warn, or error | info    |
| Log Message | The message to log                                                               |         |
