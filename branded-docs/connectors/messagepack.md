## Actions

### Decode

Decode a MessagePack message into an object

| Input   | Comments                      | Default |
| ------- | ----------------------------- | ------- |
| Message | Messagepack message to decode |         |

### Encode

Encode an object into MessagePack

| Input | Comments                | Default |
| ----- | ----------------------- | ------- |
| Data  | Data / object to encode |         |
